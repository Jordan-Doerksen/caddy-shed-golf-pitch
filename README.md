# Caddy Shed Golf — Website Pitch

Three website design directions for **Caddy Shed Golf Stores**, Winnipeg (custom-fit golf shop, family-owned since 1982). `index.html` is a hub page that links to all three so the client can compare them side by side.

> Concept & design — a portfolio pitch. Not the official Caddy Shed site.

## Pages

| File | What it is |
|---|---|
| `index.html` | **Hub / chooser** — links to all three directions + the proposal |
| `premium-heritage.html` | **Direction 01 — Premium Heritage.** Refined, editorial, private-club feel |
| `modern-sport.html` | **Direction 02 — Modern Sport.** Bold dark performance brand |
| `local-warm.html` | **Direction 03 — Local & Warm.** Friendly, family, community-first |
| `original-concept.html` | The first single-page draft the three grew out of |
| `pitch.html` | The written proposal (what's included, pricing, next steps) |

All pages are single self-contained HTML files — no build step, no dependencies. Responsive, accessible, `prefers-reduced-motion` respected, tap-to-call, embedded map, today's-hours highlight.

## Run locally

Just open `index.html` in a browser, or serve the folder:

```
python -m http.server 8080
```

## Hosting

Designed for static hosting (e.g. GitHub Pages). Set the repo's Pages source to the `main` branch root; `index.html` is the entry point.
